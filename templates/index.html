How to get a Gmail App Password

1. Go to myaccount.google.com
2. Click Security in the left sidebar
3. Enable 2-Step Verification
4. Scroll to App Passwords at the bottom
5. Type a name (e.g. "MailSender") → Create
6. Copy the 16-character password → paste it above
Gmail SMTP limits: ~500/day (personal), ~2,000/day (Workspace).
Sheet sharing: open each sheet → Share → "Anyone with the link" → Viewer.
